JavaScript Tracker Mobile (Beta)

Description

Tracker is a free video analysis and modeling tool built on the Open Source Physics (OSP) framework. It is designed to be used in education and is supported by the AAPT-ComPADRE National Science Digital Library. If you're new to Tracker, please see Help Getting Started for a step-by step beginner's guide or Getting Started with Tracker for a video tutorial before using the JavaScript version of Tracker. Additional information about Tracker is available on the main Tracker website.

Note that users can reposition the JavaScript Tracker window by dragging the title bar and that the window can be resized by dragging the lower right hand corner.

Bug Reports

  • Use the OSP Tracker Discussion forum for all other Tracker related questions or feedback. We welcome questions about how to use Tracker, suggestions for additional Tracker features, and bug reports related to the Java version of Tracker.

Video Clip Formats and Tracker Experiments

JavaScript Tracker supports video clips in the following formats:

  • Image stacks consisting of *.jpeg images numbered sequentially. For example: car00.jpg, car01.jpg, car02.jpg ........
  • Video types supported by the HTML 5 〈video〉 element, such as mp4 or webm.
Tracker Experiment files are special Tracker zip (e.g., *.trz files) archives that contain video clips, Tracker xml data files (e.g., *.trk files) and supplemental documents, such as PDF files. Experiment files that contain older video formats, such as *.qt or *.avi, can be loaded into Tracker but they will not display any video. Users will need to convert older formats into a supported format using a third-party program, such as Cloud Convert..

Another way to Tracker Experiment files or video clips into Tracker is to drag and drop them from the desktop onto the Tracker window.

Finally, users can use the Tracker library browser to import Tracker video experiments from various digital libraries, such as the OSP Collection on the AAPT-ComPADRE library -- provided that the video clip in the *.trz file is encoded in a supported format.
