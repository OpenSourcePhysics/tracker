Data Tool Online

Data Tool is a data analysis program from Open Source Physics (OSP). It is bundled with Tracker and EJS but can also be used here as a standalone program.

You can reposition Data Tool by dragging the title bar and resize it by dragging the lower right hand corner.

Data Tool can open text data files with the following formats:

  • XML files with data generated by Tracker and other OSP programs.
  • Comma-delimited text files. Each line in the text file contains a row of numeric values separated by commas. The first line may also define variable names. For more information see Data Tool Help.

To open your own data files, drag and drop them onto the Data Tool window. Or choose File|Open or File|Import and navigate to the file using the file chooser. Here are some data files you can download for testing:

Data Tool was written by Douglas Brown at Cabrillo College using portions of the Open Source Physics Java code library developed by Wolfgang Christian at Davidson College. Data Tool was later converted from Java to JavaScript by Douglas Brown, Wolfgang Christian and Robert Hanson using the SwingJS system developed by Hanson and his students at St. Olaf College. This web page runs the JavaScript conversion.
