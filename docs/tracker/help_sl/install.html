Namestitev

Tracker zahteva Javo nudi pa lasten odprtokodni video pogon  Xuggle. QuickTime je še vedno podpiran, ne pa zahtevan.

1. Preverimo oziroma namesti Javo

  • Preverimo, če je na računalniku nameščena Java verzija 1.5 ali novejša. Če nismo prepričani ali če java ni nameščena, nadaljujemo z naslednjim korakom.
  • Naložimo si najnovejši namestitveni program Jave, ki ga dobimo na http://java.sun.com/javase/downloads/. Če sami nismo razvijalci, nam zadošča JRE (Java Runtime Environment).
  • Z dvoklikom sprožimo namestitveni program in sledimo navodilom.

2. Namestitev programaTracker (z Xuggle)

  • Naložimo si namestitveni program Tracker, primeren za naš računalnik (Windows 2000/XP/Vista/7, Mac OSX, ali Linux), ki ga dobimo na http://physlets.org/tracker/.
  • Dvokliknemo na namestitveni program in sledimo navodilom. Za največjo funkcionalnost preverimo, če smo v namestitev vključili tudi Xuggle.
  • Preberemo tekstovno datoteko Tracker_README, kjer zasledimo več o sistemskih zahtevah, Nameščeni zagonski program  Tracker.exe omogoča prilagajanje zagonskega procesa za Tracker, vključno s tem, katero jar verzijo moramo pognati, kateri javanski virtualni stroj uporabiti in tudi katere druge zunanje programe.

3. Nadgradnja programa Tracker

  • Nadgradnja tracker.jar. Iz domače strani Tracker naložimo najnovejšo verzijo tracker.jar in z njo nadomestimo obstoječo, ki je v domačem direktoriju  Tracker (Program Files/Tracker ali ekvivalentni). Naložimo lahko tudi verziji specifične jar datoteke (tracker-4.xx.jar) in jih dodamo k datotekam naše verzije.
  • Nadgradnja namestitvenega programa. Naložimo nadgradnjo programa "installer" in sledimo navodilom. Nadgradnja namestitvenega programa lahko nagredi programe Tracker, Tracker Starter, Xuggle oziroma druge komponentes.

4. Zagon programa Tracker

  • V zagonskem nenuju zberemo postavko Tracker (velja le za Windows).
  • V direktoriju s poskusi in videi dvokliknemo na bližnjico (privzeto My Documents/Tracker ali ekvivalentno).
  • Na domači strani Tracker kliknemo na povezavo Web Start Tracker. Web Start vedno sproži zadnjo verzijo in ne zahteva uporabe programa Tracker Starter.
  • V domačem direktoriju Tracker dvokliknemo na datoteko Tracker.exe.
  • V domačem direktoriju Tracker (ali kakšnem drugem) dvokliknemo na datoteko tracker.jar ali tracker-x.xx.jar.

Čestitamo! Za uporabo programa Tracker si oglejmo Pa začnimo.
