SwingJS distribution: resources directory

This directory can be used to hold files that code within this Eclipse project 
needs to run -- data files, for instance. Files in it will be copied to 
the SwingJS JavaScript library within the site by by an ant script, such as build-site.xml.

The default site is the Eclipse project's ./site directory.

The default location for the SwingJS JavaScript library is: ./site/swingjs/j2s.
